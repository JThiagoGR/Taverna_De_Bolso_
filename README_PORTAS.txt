Sistema de Portas

No layout do Mestre:
- O botão do lápis agora alterna entre: parede reta, traço livre, círculo e porta.
- Quando o botão mostrar 🚪, desenhe uma linha como porta.
- A porta nasce fechada.
- Com a ferramenta de mover ativa, o Mestre clica/toca na porta para abrir/fechar.

Como funciona:
- Porta fechada bloqueia movimento dos tokens.
- Porta aberta não bloqueia.
- Jogadores não veem portas nem paredes.
- Mestre vê portas: amarelo = fechada, verde tracejado = aberta.
- Ao salvar/importar cena, portas também são salvas junto com mapa, paredes e NPCs.
