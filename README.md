## Upgrade visual estilo Demeo

Adicionado sistema visual de tokens tipo miniatura/action figure:
- PNG em pé ancorado no pé/base.
- Sombra/base no chão.
- Glow de seleção.
- Barra de vida.
- Nome acima do personagem.
- Botão no painel de token para alternar: Miniatura em pé / Top-down.

Para melhor resultado, use PNG transparente do personagem inteiro.
